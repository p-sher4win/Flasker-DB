# PYTHON FLASK PROJECT (USER CURD)

About This Project:
- Tech Stack: Python, Flask, MySQL, BootStrap
- Build an API
- Responsive UI With BootStrap
- Error Handling

### Run this app locally
1. Install dependencies
```bash
pip install -r requirements.txt
```
2. Start the app
```bash
python main.py
```
